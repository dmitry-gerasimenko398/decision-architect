Assumptions & audit trail

What this result depends on

User-confirmed assumptions

  • Ratings are net utility, so there is no separate penalty for experimenting.
  • Each action consumes exactly one remaining visit.
  • A newly tried dish's rating is received immediately on that visit and remains known for later visits.
  • The reusable best-known rating updates to the maximum of its old value and the newly observed rating.
  • There is no discounting of later visits.
  • Each untried dish is an independent draw from one fixed triangular rating distribution.
  • Observing one dish does not update the rating distribution of the remaining untried dishes.

Warnings

  • Continuous expectations use deterministic midpoint quantile quadrature, so values are numerical approximations.

Mathematical details

The report displays stored dynamic-programming output; it does not rerun the model.

V(0, u, b) = 0

Exploit:
b + V(t − 1, u, b)

Explore:
E[X + V(t − 1, u − 1, max(b, X))]

Optimal value:
max(exploit, explore)

The expectation uses deterministic midpoint quantile quadrature with the recorded point count. The engine memoizes best-known values at the recorded decimal precision and compares actions using the recorded tolerance.

Assumptions include no discounting, no learning of the distribution, and independent new-option draws from one triangular distribution. Exploration receives the new draw immediately. This is a Feynman-inspired finite-horizon model, not a claim of perfect historical reproduction.
