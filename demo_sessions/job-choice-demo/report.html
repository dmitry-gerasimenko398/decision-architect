Mathematical details

The report displays stored engine results; it does not run these calculations.

Anchor normalization:
utility = clamp((value − worst anchor) / (best anchor − worst anchor), 0, 1)

Weighted utility:
total utility = Σ (criterion weight × normalized criterion utility)

Triangular raw mean:
mean = (minimum + most likely + maximum) / 3

Monte Carlo ranking draws each confirmed triangular estimate with the recorded seed, calculates weighted utility, and ranks feasible alternatives by mean utility. Simulation ties split win credit.

One-at-a-time fixed-sample sensitivity:
w_target(x) = x
w_other(x) = original w_other × (1 − x) / (1 − original w_target)

The engine reuses stored criterion means, solves linear winner crossings analytically, and verifies both sides. It does not resample or change hard-constraint eligibility.
